# python-life-and-ministery

## Imprimir el programa del próximo mes

Ejecutar fuera de la interfaz gráfica:

```bash
python print_meeting_program.py
```

El comando consulta WOL con la misma lógica de fechas que la aplicación e imprime todas las semanas cuyo lunes pertenece al próximo mes. Para probar una fecha concreta:

```bash
python print_meeting_program.py --date 2026-06-20
```

## Imprimir los hermanos registrados

```bash
python print_participants.py
```

El comando separa hombres y mujeres, los ordena primero por título y luego alfabéticamente, y muestra las condiciones especiales activas de cada persona.
